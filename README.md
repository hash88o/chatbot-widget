# Chatbot Widget

Embeddable chat widget for any website.

## Tech Stack

- React 18
- Vite 5
- CSS Custom Properties (no UI libraries)

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Usage

```jsx
import { ChatWidget } from './components/ChatWidget';

<ChatWidget brandName="Your Company" />
```
